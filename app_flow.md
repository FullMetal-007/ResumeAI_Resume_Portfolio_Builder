## User Flow

### New User Journey

Landing page Sign up Email verification Dashboard introduction Choose Resume or Portfolio

### Resume Flow

Dashboard Resume Select Template or Custom Mode

### If Template

Fill structured form Live preview Customize theme Download or save

### If Custom Mode

Upload resume or fill profile Select role Paste job description Select experience level Generate AI optimized resume View **ATS** score Edit Download

### Portfolio Flow

Dashboard Portfolio Select Template or Agent Mode

### If Template

Choose layout Edit sections Preview Download source

### If Agent Mode

Answer structured questions Generate full portfolio Preview sandbox Edit inline Regenerate section Download source Deploy

### Deployment Flow

Download source Or Connect GitHub Push repository Deploy to Vercel Receive live **URL**
